Пояснительная записка (Cyber store)

Проект собой представляет интернет магазин, разработанный в рамках семестрового проекта по курсу «Веб-программирование».

Стек технологий: <br/>
1) PHP
2) PostgreSQL
3) NGINX
4) DOCKER

Также был задействован PHP - фреймворк Yii2.
